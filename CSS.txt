

Propiedades De CSS


margin: numero;: El margen es el espacio 
que rodea un elemento y se usa para separar 
elementos entre sí.

padding: numero;: Esta regla establece el 
relleno (padding) de todos los elementos, 
el relleno es el espacio dentro del elemento, 
entre el contenido y el borde.

box-sizing: border-box;: Esta regla establece 
el modelo de caja de todos los elementos en la 
página como "border-box". El modelo de caja 
determina cómo se calcula el ancho y alto de un 
elemento, y "border-box" significa que el ancho y 
alto incluirán el contenido, el relleno y el borde, 
en lugar de solo el contenido (el valor predeterminado).

text-decoration: none;: Esta regla elimina la decoración 
de texto subrayado que se aplica por defecto a los hipervínculos 
(<a>). En otras palabras, quita los subrayados de los enlaces 
para que no parezcan enlaces.

font-family: sans-serif;: Esta regla establece la fuente (tipografía) 
predeterminada para el texto en la página como "sans-serif". Las fuentes 
"sans-serif" son fuentes que no tienen remates decorativos en las letras, 
lo que las hace limpias y fáciles de leer en pantalla.

*: Este selector CSS asterisco (*) selecciona todos los elementos HTML 
en la página web. En otras palabras, estas reglas CSS se aplicarán a todos 
los elementos dentro del documento HTML.

position: fixed;: Esto establece la posición del elemento <header> como "fixed", 
lo que significa que el encabezado se mantendrá en la misma posición en la ventana 
del navegador incluso si se desplaza hacia abajo en la página. Es común utilizar 
esta propiedad para encabezados que deben permanecer visibles mientras se desplaza 
por una página.

top: 0;: Establece la distancia desde la parte superior de la ventana del navegador a 
la parte superior del elemento <header>. En este caso, se coloca el encabezado en la 
parte superior de la ventana del navegador.

left: 0;: Establece la distancia desde el lado izquierdo de la ventana del navegador a la 
izquierda del elemento <header>. Esto asegura que el encabezado esté alineado con el borde 
izquierdo de la ventana del navegador.

width: 100%;: Establece el ancho del elemento <header> al 100% del ancho de la ventana del 
navegador, lo que significa que el encabezado se extenderá horizontalmente para cubrir toda la 
ventana.

height: 80px;: Establece la altura del elemento <header> en 80 píxeles. Puedes ajustar este 
valor según tus necesidades para determinar la altura del encabezado.

background: #fff;: Establece el color de fondo del elemento <header> en blanco (#fff). Puedes 
cambiar "#fff" a cualquier otro valor de color que desees.

box-shadow: 0 4px 25px -22px black;: Agrega una sombra al elemento <header>. Esta sombra tiene un 
desplazamiento horizontal de 0, un desplazamiento vertical de 4 píxeles, un radio de desenfoque 
de 25 píxeles y un esparcimiento de -22 píxeles. La sombra es de color negro.

z-index: 2;: Establece el índice de apilamiento (z-index) del elemento <header> en 2. Esto controla 
la posición del encabezado en relación con otros elementos en la página. Un valor más alto de z-index 
significa que el encabezado se superpondrá a elementos con un valor de z-index más bajo.

max-width: 1200px;: Esta propiedad establece el ancho máximo del elemento .header-content en 1200 píxeles. 
Esto significa que el contenido dentro del encabezado no se expandirá más allá de ese ancho máximo, lo que 
puede ser útil para controlar la anchura máxima del contenido del encabezado.

margin: auto;: La propiedad margin se utiliza para centrar horizontalmente el elemento .header-content dentro 
de su contenedor. Al establecer margin en "auto", el navegador calculará automáticamente los márgenes izquierdo 
y derecho iguales para centrar el contenido.

display: flex;: Esta propiedad se utiliza para establecer el modelo de diseño de caja de flexbox en el elemento 
.header-content. Flexbox es un sistema de diseño que permite organizar y alinear elementos en un diseño flexible 
y responsivo.

justify-content: space-between;: Esta propiedad de flexbox se aplica a los elementos secundarios (hijos) del elemento .header-content. justify-content con el valor "space-between" distribuirá los elementos secundarios horizontalmente 
de manera que haya espacio igual entre ellos y el espacio restante a los extremos del contenedor. Esto a menudo se 
usa para alinear elementos a los extremos izquierdo y derecho del contenedor, creando un diseño de encabezado con 
elementos a ambos lados.
