

Propiedades De HTML


<!DOCTYPE html>: Declaración doctype que define la 
versión de HTML utilizada (HTML5 en este caso).

<html lang="es">: Elemento HTML raíz que indica el 
inicio del documento HTML. El atributo lang se utiliza 
para especificar el idioma del documento (en este caso, español).

<head>: Sección de encabezado que contiene metadatos y 
enlaces a recursos externos. Aquí se encuentra la 
configuración de codificación, la declaración del título de la 
página y otros metadatos.

<meta charset="UTF-8">: Declara la codificación de caracteres 
UTF-8 para admitir caracteres especiales y emojis.

<meta name="viewport" content="width=device-width, initial-scale=1.0">: 
Configura la escala inicial y la visualización en dispositivos móviles.

<title>Título de la página</title>: Define el título de la página 
que se mostrará en la pestaña del navegador.

<body>: Contiene el contenido visible de la página, incluyendo encabezados, 
párrafos, imágenes, enlaces y más.

<header>: Sección que generalmente contiene el encabezado de la página, 
como el logotipo y la navegación.

<main>: Sección que contiene el contenido principal de la página, como artículos, 
textos y otros elementos importantes.

<footer>: Sección que contiene información de pie de página, como derechos de 
autor y enlaces a redes sociales.

<dl>: Esta etiqueta es la envoltura principal de la lista de definición. Se utiliza 
para indicar el inicio de la lista de definición.

<dt>: Esta etiqueta se utiliza para definir un término o un encabezado dentro de la 
lista de definición. Se coloca dentro de la etiqueta <dl>. Los términos son generalmente 
texto en negrita.

<dd>: Esta etiqueta se utiliza para proporcionar la definición o descripción del término 
definido en <dt>. Se coloca después de <dt> y contiene el texto que describe el término.

<tr> define una fila de la tabla.

<th> define las celdas de encabezado de la tabla.

<td> define las celdas de datos de la tabla.
