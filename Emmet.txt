
Funciones para HTML

html:5 + Tab: para crear la estructura basica de html.

header>nav>ul>li*10: para crear 10 elementos en una lista desordenada dentro de un nav y un header.

section.NombreDeLaClase#NombreDelID: para crear un id y una clase dentro de una seccion.

header>nav>ul>li*10>a:link[target="_blanck"]{Menu #$}: para crear lo mencionado anteriormente, 
pero esta vez con 10 enlaces que tengan el atributo target y como contenido la palabra menu con 
los numeros del 1 al 10.

meta:vp: para crear la etiqueta de viewport.

link:css: para crear la etiqueta de enlace al archivo CSS.

link:favicon: para crear la etiqueta de poner icono.
